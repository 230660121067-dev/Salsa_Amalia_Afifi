Deskripsi Kursus

Kursus ini dirancang bagi **pemula total** yang ingin memahami konsep dasar pemrograman. Kami akan mulai dari nol, memperkenalkan logika komputasi, struktur data dasar, dan sintaksis dari dua bahasa paling populer di dunia: **Python** dan **JavaScript**. Setelah menyelesaikan kursus ini, Anda akan siap untuk beralih ke pengembangan web, analisis data, atau bidang spesialisasi pemrograman lainnya.

Apa yang Akan Anda Pelajari?

  • Memahami konsep dasar algoritma dan *flowchart*.
  • Menguasai sintaksis dasar Python (variabel, perulangan, kondisi).
  • Menggunakan JavaScript untuk manipulasi elemen web sederhana (DOM).
  • Bekerja dengan struktur data dasar seperti list, dictionary, dan array.
  • Menulis kode yang bersih, terdokumentasi, dan efisien.

Struktur Modul (Pelajaran)

  • Modul 1: Pengantar Komputasi dan Algoritma (2 jam)
  • Modul 2: Dasar-Dasar Python (Variabel & Tipe Data) (3 jam)
  • Modul 3: Struktur Kontrol (If/Else & Looping) (4 jam)
  • Modul 4: Struktur Data Python Lanjut (List, Tuple, Dictionary) (3 jam)
  • Modul 5: Pengantar JavaScript (Sintaksis Dasar) (4 jam)
  • Modul 6: Interaksi Web Sederhana dengan DOM (JavaScript) (2 jam)
  • Modul 7: Proyek Akhir: Membuat Kalkulator Sederhana (2 jam)
